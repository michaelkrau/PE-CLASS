# שיעור חינוך גופני
בשיעור חינוך גופני עם תלמידות משתי כיתות י' שלומדות יחד שיחקנו כדורעף. המשתתפות המרכזיות באירוע היו אני, המורה לחינוך גופני, ונועה מכיתה י'2, לצד שאר התלמידות שהתחלקו לקבוצות ושיחקו כרגיל על המגרש.

במהלך המשחק שמתי לב שנועה מתנהגת בצורה שחזרה על עצמה שוב ושוב. בכל פעם שהכדור הגיע לכיוונה היא זזה הצידה או קפאה במקום, ונתנה לכדור ליפול בלי לנסות לגעת בו. לאורך רוב המשחק היא כמעט לא השתתפה במהלך עצמו, וכשהכדור נפל לידה הבנות בקבוצה שלה התעצבנו וצעקו עליה שהיא הורסת להן את המשחק.
