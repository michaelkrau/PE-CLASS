# שיעור כדורעף
בשיעור הכדורעף עם כיתה י׳ פתחתי בחימום קצר ואז חילקתי את התלמידות לקבוצות לתרגול מסירות, קבלה והנחתה. כבר אחרי כמה דקות ראיתי איך הביטחון שלהן עולה—גם מי שבתחילת השיעור היססה, התחילה לבקש כדור ולהוביל מהלכים. סיימנו במשחקון מסכם עם הרבה עידוד, עבודת צוות ואווירה מצוינת, ויצאתי מהשיעור בתחושה שהן לא רק השתפרו מקצועית אלא גם למדו להאמין בעצמן ובחברות שלהן.
